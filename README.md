# Gradio OpenAI Chatbot (Streaming)

Small Gradio chat app that calls OpenAI and streams the response.

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r Requirements.txt
````

Create a `.env` file:

```bash
OPENAI_API_KEY=your_key_here
```

## Run

```bash
python gradio_setup.py
```
